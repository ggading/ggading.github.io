Membuat Blog Dengan HTML Sederhana

Dibuat tanggal: 29-09-2022

Saya nggak akan menjelaskan tentang bagaimana saya membuat website ini karena kalian bisa lihat di source html. Tapi saya akan mejelaskan setup saya sekarang dan setup yang saya akan pakai.

Hosting dan Domain

Saya memilih GithubPage sebagai hostingnya. Di kedepannya saya mungkin akan host di Domainesia. Email masih di Google tapi saya juga berencana dapat email sendiri.

Penulisan

Saya edit langsung HTMLnya dengan vim/emmet.vim di Linux. Saya berencana menggunakan Hugo kedepannya.

Estetik

Tidak ada ( :V ) selain meggunakan <meta name="color-scheme" content="dark light"> dan <body style="font-family:sans-serif">
